## Запуск
Must be installed:
* Maven 3
* Java 8
* PostgreSQL 9.5

1. Install DB dumps from
```db/dump/*```

2. Setup config path and JDBC datasource in
```config/web-context-dev.xml```

3. Start 
```mvn tomcat:run```
